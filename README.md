# Power BI Sales Dashboard

## Overview
This project is a Power BI dashboard created to analyze sales performance and key business metrics.
It focuses on turning raw data into clear, actionable insights for decision-making.

## Tools Used
- Power BI
- Excel / CSV
- Data cleaning and basic DAX

## Key Insights
- Sales trends over time  
- Top-performing products  
- Performance by region or category  

## Files in This Repository
- `.pbix` file – Power BI dashboard
- Dataset file – source data used for analysis
- Screenshot(s) – dashboard preview

## How to Use
1. Download the `Project 1_The Executive Sales & Operations Overview.pbix` file  
2. Open it using **Power BI Desktop**
3. Interact with filters and visuals
